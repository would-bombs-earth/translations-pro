# 隐私与用户控制规范

## 默认行为

- 翻译历史功能**默认启用**
- 所有数据仅存储在用户本地浏览器 IndexedDB 中
- **不会**将历史数据发送到任何远程服务器

## 用户控制

### 禁用历史记录

在"翻译历史"面板中增加一个切换开关，允许用户完全关闭历史记录。

关闭后：
- 不再记录新的翻译事件
- 已有历史数据保留（不清除）
- 用户仍可查看和导出已有历史

### 清空历史

- "清空历史"按钮需要二次确认
- 确认对话框使用浏览器原生 `confirm()`
- 清空后不可恢复

### 单条删除

- 每条历史条目右侧提供"✕"删除按钮（与排除域名列表交互一致）
- 点击后立即从 IndexedDB 删除该条目，无需确认

## 数据安全

- 不记录包含用户身份信息的内容（不主动收集，但翻译内容本身由用户决定）
- 不设置自动过期删除（由容量限制 5000 条兜底）
- 导出 JSON 文件中包含翻译内容，用户自行保管

## 合规提示

在首次启用时（第一次显示历史面板时）提示用户：

> "翻译历史仅保存在本地浏览器中，不会上传到任何服务器。你随时可以清空或关闭记录。"
